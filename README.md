UPI 支付流程与系统层问题

在真实业务中，UPI 支付常见问题包括：

钱包显示成功 ≠ 系统确认成功

回调延迟或丢失

UTR 生成时间不一致

高并发下状态判断混乱

因此，单纯依赖回调或前端结果，往往无法满足稳定运营需求。

本研究重点关注：

交易状态迁移模型

延迟确认与补偿机制

多维度交易匹配思路
